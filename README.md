# 🌤️ Application Météo

## Description
Application web qui affiche la météo en temps réel pour n'importe quelle ville.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript (ES6)
- API OpenWeatherMap

## Fonctionnalités principales
- Recherche météo par nom de ville
- Affichage : température, ressenti, vent, humidité
- Fond d'écran dynamique selon la météo
- Animation d'apparition des cartes

## Lien GitHub Pages
[https://Louay-hamed.github.io/app_meteo](https://Louay-hamed.github.io/app_meteo)

## Nouveautés explorées
- Utilisation d'une API REST avec fetch() et async/await
- Manipulation dynamique du DOM
- Sauvegarde avec localStorage

## Difficultés rencontrées
- Comprendre le fonctionnement asynchrone de JavaScript
- Gérer les erreurs de l'API (ville non trouvée)
- Changer le fond selon la condition météo

## Solutions apportées
- Utilisation de async/await pour attendre les réponses API
- Ajout d'un message d'alerte si la ville n'existe pas
- Fonction changerFond() avec conditions if/else
